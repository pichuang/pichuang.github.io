йҖІж“Ҡзҡ„ OpenAPI!!!

Posted on 2018-07-23 | Edited on 2019-03-25 | In openapi

OpenAPI Specification, OAS

йҡЁи‘—иіҮиЁҠжңҚеӢҷзҡ„йЈӣеўһеҸҠиӨҮйӣңпјҢеҗ„е®¶е» е•ҶжүҖжҸҗдҫӣзҡ„еә•еұӨжңҚеӢҷе°ҒиЈқеҫҢжҸҗдҫӣеҮәдҫҶзҡ„д»ӢйқўпјҢжҲ‘еҖ‘зЁұеҒҡ API (Application Program Interface)пјҢиҖҢ OpenAPI еүҮжҳҜе°ҮйҖҷдәӣ API дҫқж“ҡжҹҗдәӣиҰҸзҜ„е…¬й–ӢеҮәдҫҶзөҰз¬¬дёүж–№дҪҝз”ЁпјҢеҸҜеҢ…еҗ«е…ҚиІ»дҪҝз”ЁгҖҒд»ҳиІ»дҪҝз”ЁжҲ–йҷҗе®ҡдҪҝз”Ёзӯүж–№ејҸпјҢжңүдёҖеҖӢе…¬й–Ӣж—ўе®ҡзҡ„иҰҸзҜ„еҸҜдҫӣеҸғиҖғгҖӮ

иӘ°жҳҜ API жҸҗдҫӣиҖ…?

д»Ҙзӣ®еүҚзңӢеҲ°зҡ„жЎҲдҫӢжңҖеӨ§е®—зҡ„е°ұжҳҜ Cloud Service Provider (CSP) еҢ…еҗ«еғҸ GoogleгҖҒAmazonгҖҒMicrosoft зӯүзӯүпјҢд»–еҖ‘зҡ„е…·й«”еҒҡжі•жҳҜе°Үз”ўе“ҒеҠҹиғҪзӣЎеҸҜиғҪзҡ„ API еҢ–еҫҢжҸҗдҫӣзөҰй–Ӣзҷјдәәе“ЎеҒҡдәҢж¬Ўй–ӢзҷјпјҢиӘһжі•иҰҸзҜ„йғҪеҖӢеҲҘз”ұ CSP жҸҗдҫӣпјҢиӯ¬еҰӮеӨ§е®¶йғҪзҶҹзҹҘзҡ„ Google Map APIгҖӮ

иҖҢз¬¬дәҢеӨ§йЎһзҡ„е°ұеұ¬йҮ‘иһҚжҘӯпјҢеҗ„е®¶йҠҖиЎҢжң¬иіӘжҸҗдҫӣзҡ„е…§е®№йғҪдёҖжЁЈпјҢдҪҶжңҚеӢҷж–№ејҸеҚ»еӨ§дёҚзӣёеҗҢпјҢзҸҫеңЁз§‘жҠҖйҖІжӯҘеҲ°дҪҝз”ЁиҖ…зҝ’ж…ЈеӨ§зӮәи®ҠеҢ–пјҢеҫһйҒҺеҺ»и¶•дёүй»һеҚҠеҲ°зҸҫеңЁеҸӘиҰҒз¶Іи·ҜйҠҖиЎҢз·ҡдёҠй»һдёҖй»һе°ұеҘҪпјҢиӘ•з”ҹеҗ„е®¶еӮізөұйҠҖиЎҢе°Қж–јж•ёдҪҚиҪүеһӢе’Ңеүөж–°зҡ„йңҖжұӮпјҢд№ҹе°ұжҳҜ Fintech йҮ‘иһҚз§‘жҠҖжҲ–иҖ…жҳҜ Open BankпјҢеҲ©з”Ёж•ёдҪҚеҢ–йҠҖиЎҢжҘӯеӢҷжҸҗдҫӣеҖӢдәәжҲ–жі•дәәжӣҙзҙ°з·»жҲ–жӣҙе»Јзҡ„йңҖжұӮпјҢйҖІиҖҢж“ҙеӨ§йңҖжұӮеҸҠдәӨжҳ“йҮҸгҖӮжүҖд»Ҙеүөж–°жҰӮеҝөеҸҠжңҚеӢҷжҺЁеҮәйҖҹеәҰжҳҜйқһеёёеӨ§зҡ„й—ңйҚөгҖӮйҖҸйҒҺзөұдёҖиӘһж„ҸеҢ–зҡ„ Open API еҸҜд»Ҙе№«еҠ©е…§йғЁжңҚеӢҷзӣёдә’е Ҷз–Ҡзө„еҗҲиҖҢжҲҗдёҖеҖӢеүөж–°жңҚеӢҷпјҢеҝ«йҖҹжҺЁеҮәз”ўе“ҒзҚІеҫ—еёӮе ҙеӣһйҘӢгҖӮ

жңҖеҫҢе°ұжҳҜж”ҝеәңзӣёй—ңе–®дҪҚпјҢи »еӨҡе…ҲйҖІеңӢе®¶жңҖдёҖй–Ӣе§ӢеҒҡзҡ„йғҪжҳҜ Open DataпјҢй–Ӣж”ҫиіҮж–ҷдҫӣзөҰдёҖиҲ¬дәәеҒҡдҪҝз”ЁеҲҶжһҗпјҢдҪҶеҲ°дёӢдёҖеҖӢеұӨж¬Ўе°ұиҰҒжҖқиҖғпјҢиҰҒеҰӮдҪ•жҸҗдҫӣжәҗжәҗдёҚж–·дё”еҸҜд»Ҙз¶ӯиӯ·зҡ„иіҮж–ҷдҫҶжәҗпјҢеүҮе°ұжҳҜ Open API зҡ„дәӢжғ…дәҶгҖӮеҗ„дҪҚеҸҜд»ҘеҸғиҖғжңҖе…Ҳиө·й ӯзҡ„зҫҺеңӢзҡ„з¶“й©— 2.0зҫҺеңӢж”ҝеәңй–Ӣж”ҫиіҮж–ҷжҺЁеӢ•зҸҫжіҒиҲҮеҹ·иЎҢз¶“й©—гҖӮжңүи¶Јзҡ„дәӢжғ…жҳҜпјҢзӣ®еүҚеҸ°зҒЈеңЁйҖҷж–№йқўзҡ„жҲҗзёҫжҳҜйқһеёёйЎҜи‘—ең°пјҢжң¬ж–Үж’°еҜ«жҷӮй–“ 2018/7 еҸ°зҒЈеңЁиӢұеңӢй–Ӣж”ҫзҹҘиӯҳеҹәйҮ‘жңғпјҲOpen Knowledge Foundationпјүдёӯзҡ„и©•жҜ”жҳҜз¬¬дёҖеҗҚгҖӮ

еӨ§е®¶йғҪзҹҘйҒ“жңүж•ёж“ҡиҖ…зӮәеӨ§пјҢдҪҶж•ёж“ҡжҢҒжңүиҖ…д№ҹйңҖиҰҒжҖқиҖғжҖҺйәјеҲ©з”Ёж•ёж“ҡжҸҗдҫӣеҮәеүөж–°жңҚеӢҷзөҰж¶ҲиІ»иҖ…пјҢйҖІиҖҢеҲәжҝҖиІ·е–®пјҢеңЁеёӮе ҙеӣһйҘӢдёҚжҳҺзўәзҡ„зӢҖжіҒдёӢпјҢеҲ©з”Ё Agile ж•ҸжҚ·й–ӢзҷјжөҒзЁӢиҝӯд»ЈеҮәж–°жңҚеӢҷжё¬и©ҰеёӮе ҙеҸҚжҮүжҳҜзӣ®еүҚеёёиҰӢзҡ„жЁЎејҸгҖӮ

еҸ°зҒЈж”ҝеәңж–ј Open API дёҠзҡ„еҠӘеҠӣ

иҝ‘е№ҙж”ҝеәңзӮәдәҶж“ҙеӨ§иіҮиЁҠжңҚеӢҷзҡ„ж•ҲзӣҠпјҢз¶“йҒҺиіҮиЁҠж”ҝе§”е”җйііе№ҫж¬Ўзҡ„е…¬й–ӢиЁҺи«– ( 2, 3 )пјҢжңҖзөӮеңӢе®¶зҷјеұ•е§”е“ЎжңғеҸғиҖғ Linux Foundation еә•дёӢе°ҲжЎҲ Open API Initiative (OAI) жЁҷжә–е®ҡзҫ©еҮә OpenAPI Specification (OAS) дёҰзҝ»жҲҗдёӯж–Үж–Үд»¶пјҢзӣ®жЁҷжҳҜеёҢжңӣзөұдёҖж”ҝеәңж©ҹй—ңзҡ„ API иғҪж“ҒжңүдёҖиҮҙжҖ§иЎЁиҝ°пјҢи®“дәәйЎһе’Ңж©ҹеҷЁйғҪиғҪзңӢеҫ—жҮӮпјҢи©ізҙ°еүҚеӣ еҫҢжһңеҸҜд»ҘеҸғиҖғе№ҫеҖӢе°Ҹжўқж–ҮпјҢи®“ж”ҝеәңжҸҗдҫӣжӣҙеҘҪзҡ„ж•ёдҪҚжңҚеӢҷгҖӮ

зӣ®еүҚеңЁеңӢзҷјжңғзҡ„зӯ–еҠғд№ӢдёӢе·Із¶“еҖӢең°ж–№ж”ҝеәңе·Із¶“еҜҰдҪңдәҶдёҚе°‘ Open API ж”ҫиҮіж”ҝеәңиіҮж–ҷй–Ӣж”ҫе№іиҮә дёҠдҫӣзӨҫжңғеӨ§зңҫдҪҝз”ЁгҖӮиҖҢжғіиҰҒдәҶи§ЈдёҖдәӣеңӢеӨ–дҪҝз”Ё API зҡ„ж”ҝеәңдјҒжҘӯжЎҲдҫӢжҺўиЁҺпјҢеҸҜд»ҘеҸғиҖғеүҚйҷЈеӯҗиҲүиҫҰзҡ„ 2018-06-28 Red Hat ж•ёдҪҚеүөж–°жҠҖиЎ“еә§и«Үжңғ - Saylt зҡ„йҖҗеӯ—зЁҝе…§е®№пјҢдәҶи§ЈеңӢеӨ–еҜҰйҡӣзҷјз”ҹзҡ„зӢҖжіҒгҖӮ

е°Қж–је…¬йғЁй–ҖеҸҠе» е•ҶдҫҶиӘӘпјҢжңүдёүд»Ҫж–Үд»¶е°ұи®Ҡеҫ—еҫҲйҮҚиҰҒдәҶ

  1. е…ұйҖҡжҖ§жҮүз”ЁзЁӢејҸд»ӢйқўиҰҸзҜ„ - еңӢе®¶зҷјеұ•е§”е“Ўжңғ дё»з®Ўжі•иҰҸжҹҘи©ўзі»зөұ

  2. 1060911 дҝ®жӯЈзёҪиӘӘжҳҺеҸҠжўқж–Үе°Қз…§иЎЁ - ж©ҹй—ңе§”иЁ—иіҮиЁҠжңҚеӢҷе» е•Ҷи©•йҒёеҸҠиЁҲиІ»иҫҰжі•

    з¬¬дёғжўқ з¬¬еҚҒдәҢж¬ҫ ж©ҹй—ңжҺЎиіји»ҹй«”й–ӢзҷјжңҚеӢҷпјҢеүҚй …з¬¬дәҢж¬ҫжүҖе®ҡе» е•Ҷд№Ӣе°ҲжҘӯжҠҖиЎ“иғҪеҠӣпјҢеҫ—еҢ…жӢ¬еңЁйӣ¶жҲҗжң¬жҲ–дҪҺжҲҗжң¬д№ӢеүҚжҸҗдёӢпјҢжҸҗдҫӣеҸҜиҮӘз”ұеӯҳеҸ–гҖҒдҪҝз”ЁгҖҒдҝ®ж”№еҸҠж•Јеёғд№Ӣе…ұйҖҡжҖ§жҮүз”ЁзЁӢејҸд»Ӣйқўй–ӢзҷјжҲ–ж•ҙеҗҲиғҪеҠӣгҖӮ

  3. [1051206 иіҮиЁҠжңҚеӢҷжҺЎиіјпӨүзҙ„зҜ„жң¬з¬¬ 2 жўқеҸҠз¬¬ 5 жўқдҝ®жӯЈиҚүжЎҲжўқж–Үе°Қз…§иЎЁ][9]

    з¬¬дәҢжўқ п§ҹзҙ„жЁҷзҡ„ з¬¬еӣӣж¬ҫ п§ҹзҙ„жЁҷзҡ„ж¶үеҸҠжҮүз”ЁзЁӢејҸд»Ӣйқўй–ӢзҷјжҲ–еўһдҝ®иҖ…пјҢжҮүдҫқеңӢе®¶зҷјеұ•е§”е“ЎжңғиЁӮй ’д№ӢжңҖж–°зүҲгҖҢе…ұйҖҡжҖ§жҮүз”ЁзЁӢејҸд»Ӣйқўй–Ӣж”ҫиҰҸзҜ„гҖҚиҫҰп§ӨпјҢдёҰйҒӢз”ЁеңӢйҡӣйҖҡз”Ёй©—иӯүж©ҹеҲ¶пјҲеҰӮ Linux Foundation д№Ӣ OpenAPI зӯүпјүпјҢдҪңзӮәй©—ж”¶д№Ӣдҫқж“ҡгҖӮ

йҖҸйҒҺеҫһеҲ¶еәҰжЁҷжә–йқўдёҠпјҢзӣҙжҺҘиӘҝж•ҙжі•иҰҸпјҢи®“еҫҢзәҢзҡ„ж”ҝеәңиіҮиЁҠзі»зөұе»әзҪ®жңүжүҖжі•жәҗеҸҜд»ҘдҫҶе»әз«Ӣ API жңҚеӢҷпјҢд»Ҙзӣ®еүҚдҫҶзңӢйҖҷжҳҜжҜ”ијғжңүж•Ҳй©…еӢ•ж”ҝеәңйҖІиЎҢж•ёдҪҚиҪүеһӢзҡ„жңүж•Ҳж–№ејҸд№ӢдёҖгҖӮ

Deep Dive to OpenAPI

еҰӮдҪ•еҜ«еҮәз¬ҰеҗҲ OAS зҡ„ж–Үд»¶?

еҹәжң¬дёҠйғҪжҳҜйҒөз…§ OpenAPI Initiative жүҖйҮӢеҮәзҡ„ж јејҸжүҖж’°еҜ«пјҢеҸҜеҸғиҖғ OAI/OpenAPI-Specification - GitHubпјӣиӢҘжҳҜйҮқе°ҚеҸ°зҒЈе…¬йғЁй–ҖжүҖйңҖзҡ„иҰҸзҜ„еүҮеҸҜеҸғиҖғ е…ұйҖҡжҖ§жҮүз”ЁзЁӢејҸд»ӢйқўиҰҸзҜ„ - еңӢе®¶зҷјеұ•е§”е“Ўжңғ дё»з®Ўжі•иҰҸжҹҘи©ўзі»зөұ жүҖе®ҡзҫ©д№Ӣе…§е®№зӮәеҹәжә–пјҢеӨ§йҮҸдҪҝз”ЁжЎҲдҫӢд№ҹеҸҜеҸғиҖғ ж”ҝеәңиіҮж–ҷй–Ӣж”ҫе№іиҮә ж”ҫеңЁдёҠйқўзҡ„е…§е®№пјҢеј·зғҲе»әиӯ°иҰҒи®ҖдёҖдёӢгҖӮ

й©—иӯү OAS ж јејҸжӯЈзўәиҲҮеҗҰ

й–ӢзҷјиҖ…еҜ«еҮәдҫҶзҡ„ OAS еӢўеҝ…иҰҒз¶“йҒҺй©—иӯүжүҚзҹҘйҒ“иғҪдёҚиғҪиў«жӯЈзўәи®ҖеҸ–пјҢMermade Swagger 2.0 to OpenAPI 3.0.0 converter е°ҲжЎҲе°ұжҳҜжҸҗдҫӣй©—иӯү OAS 2.0 еҸҠ 3.0 зҡ„еҠҹиғҪпјҢзӣ®еүҚеӨ§еӨҡж•ёжҮүи©ІйғҪйӮ„жҳҜеңЁ 2.0 зҡ„зүҲжң¬жҜ”ијғеӨҡпјҢ3.0 зӣёдҝЎеңЁжңӘдҫҶзҡ„дёҚд№…жңғжҲҗзӮәдё»жөҒгҖӮ

Swagger - OpenAPI ж–Үд»¶з”ўз”ҹжңҚеӢҷ

зӣ®еүҚеёёиҰӢе°ұе…©еҖӢ Swagger еҸҠ ReDoc жңҚеӢҷпјҢеҰӮжһңжғіиҰҒжүҫжӣҙеӨҡзҡ„е·Ҙе…·жҲ–з¶Із«ҷеҸҜд»ҘеҸғиҖғ OpenAPI Tools е…§жңүдёҚе°‘ж–Үз« гҖӮеҖӢдәәдҪҝз”ЁеҫҢеҝғеҫ—йӮ„жҳҜиҰәеҫ— Swagger жҜ”ијғйҖҡз”ЁпјҢдё»еӣ жҳҜз¶Ій Ғжҳ“и®ҖеҸҠиҝ‘жңҹй–Ӣе§Ӣж”ҜжҸҙ OAS v3.0пјҢеҗ„дҪҚеҸҜд»ҘзңӢеҲ°дёӢйқўжңүжҲӘең–пјҢSwagger ж–Үд»¶жңүзү№еҲҘжЁҷиЁҳ OAS3 ж”ҜжҸҙгҖӮ

дҪҝз”Ёж–№ејҸеҫҲз°Ўе–®пјҢд»Ҙ Swagger зӮәдҫӢ

  1. й»һй–Ӣ petstore.swagger.io
  2. е°Үз¬ҰеҗҲ OAS v3.0 иҰҸзҜ„зҡ„жӘ”жЎҲж”ҫеҲ°з©әзҷҪж¬„дҪҚ
  1. й»һйҒё Explore еҚіеҸҜзҚІеҫ—дёҖд»Ҫ OpenAPI иӘӘжҳҺж–Үд»¶

еҰӮдҪ•з®ЎзҗҶйҖҷдёҖе Ҷ OpenAPI ?

еҲ°йҖҷйӮҠзӮәжӯўпјҢеҗ„дҪҚеҸҜд»Ҙеҫһ Swagger ж–Үд»¶е…§и«ӢжұӮдёҚеҗҢзҡ„иіҮжәҗзҚІеҫ—е°ҚжҮүзҡ„иіҮж–ҷгҖӮз„¶иҖҢ OpenAPI зҡ„иЁӯиЁҲеӨҡж•ёзӣҙжҺҘжҸӯйңІиіҮж–ҷзөҰеӨ–йғЁйҖІиЎҢеӯҳеҸ–пјҢеҖҳиӢҘзі»зөұдёҖеӨҡпјҢзӣёдә’зі»зөұе‘јеҸ«зҡ„ж¬Ўж•ёи¶ҠеҠ й »з№ҒпјҢз®ЎзҗҶжңғи®ҠжҲҗеҫҲеӨ§зҡ„е•ҸйЎҢпјҢж•…йҮқе°Қ Open API зҡ„з®ЎзҗҶдёҠпјҢеүҮжңғжңүдёҖ API Gateway зҡ„иЁӯиЁҲпјҢдҫҶжӣҝе…§йғЁ Open APIs йҖІиЎҢз®ЎзҗҶеҸҠдҝқиӯ·пјҢж•…жңғжңүе°ҚжҮүзҡ„и§Јжұәж–№жЎҲдҫҶиІ иІ¬жҸҗдҫӣйҖҷж–№йқўзҡ„дҝқиӯ·пјҢдҫӢеҰӮ Red Hat 3scaleпјҢй—ңж–јеҹәжң¬зҡ„ OpenAPI Gateway зӣёй—ңзҡ„иЁӯиЁҲеҸҠеҠҹиғҪпјҢд№ӢеҫҢжңғзү№еҲҘжӢүдёҖзҜҮдҫҶд»Ӣзҙ№гҖӮ
